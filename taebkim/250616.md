**오늘의 메모(선택)**: 문제 번호 완전 크리스마스잖아?

**코드(선택)**:

```c++
#include <iostream>
#include <vector>
#include <queue>
#include <string>
#include <algorithm>
#include <iomanip>
#include <set>
#include <vector>
#include <utility>

int main() {
    std::string num1, num2;
    std::cin >> num1 >> num2;

    long long result = 0;

    for (char a : num1) {
        for (char b : num2) {
            result += (a - '0') * (b - '0');
        }
    }

    std::cout << result;
    return 0;
}
```
