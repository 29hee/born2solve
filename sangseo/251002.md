**문제 이름 또는 번호**: 9713 / Sum of Odd Sequence

**문제 링크**: https://www.acmicpc.net/problem/9713

**오늘의 메모(선택)**: 간단한 수학문제, 처음으로 영어로 된 문제 풀어봄.

**코드(선택)**:
```python
t = int(input())
for i in range(t):
    n = int(input())
    out = 0
    for j in range(n+1):
        if j % 2:
            out += j
    print(out)
```
