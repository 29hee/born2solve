**오늘의 메모(선택)**: 파이썬 딕셔너리를 잘 못다뤄 못품..

**코드(선택)**:
```python
n = int(input())
cards = list(map(int, input().split()))
dicts = {}
for i in range(n):
    if str(cards[i]) in dicts:
        dicts[str(cards[i])] += 1
    else dicts[str(cards[i])] = 1
m = int(input())
ints = list(map(int, input().split()))
for num in ints:
    print(dicts[str(num)], end=' ')
```
