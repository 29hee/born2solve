**문제 이름 또는 번호**: 11047 / 동전 0

**문제 링크**: https://www.acmicpc.net/problem/11047

**오늘의 메모(선택)**: 이지

**코드(선택)**:
```python
from collections import deque
n, k = map(int, input().split())
deq = deque()
for i in range(n):
    deq.appendleft(int(input()))
cnt = 0
for i in range(n):
    if k == 0:
        break
    cur = deq[i]
    if cur > k:
        continue
    cnt += k // cur
    k %= cur
print(cnt)
```
