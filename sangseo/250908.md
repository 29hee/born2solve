**오늘의 메모(선택)**: 아 자꾸 실수해서 결국 못푼.. 낼 마저 풀어야지..

**코드(선택)**:
```python
import sys
from collections import deque

input = sys.stdin.readline
t = int(input())
for i in range(t):
    n, p = map(int, input().split())
    deq = deque(map(int, input().split()))
    cnt = 1
    idx = p
    while (deq):
        maxdeq = max(deq)
        cur = deq.popleft()
        if cur < maxdeq:
            deq.append(cur)
            if idx == 0:
                idx = len(deq) - 1
        else:
            if idx == 0:
                print(cnt)
                break
            idx -= 1
        cnt += 1
```
