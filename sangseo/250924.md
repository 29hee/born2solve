**문제 이름 또는 번호**: 17219 / 비밀번호 찾기

**문제 링크**: https://www.acmicpc.net/problem/17219

**오늘의 메모(선택)**: 딕셔너리로 저장해서 키로 밸류 찾기하면 바로 문제해결될거같아서 파이썬 내장 딕셔너리와 메소드로 간단히 풀었다.

**코드(선택)**:
```python
n, m = map(int, input().split())
d = dict()
for i in range(n):
    a, b = input().split()
    d[a] = b
for i in range(m):
    print(d.get(input()))
```
