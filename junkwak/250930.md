**문제 이름 또는 번호**:  공 바꾸기


**문제 링크**https://www.acmicpc.net/problem/10813

**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다. 근데 어렵네요.


**코드(선택)**:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N, M; 
    if (!(cin >> N >> M)) return 0;

    vector<int> a(N + 1);
    for (int i = 1; i <= N; ++i) a[i] = i;

    while (M--) {
        int i, j; 
        cin >> i >> j;
        swap(a[i], a[j]);
    }

    for (int i = 1; i <= N; ++i) {
        if (i > 1) cout << ' ';
        cout << a[i];
    }
    cout << '\n';
    return 0;
}
