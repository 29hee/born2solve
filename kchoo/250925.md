**오늘의 메모(선택)**: 

greedy문제

**코드(선택)**: 

```c++
#include <vector>
#include <algorithm>

using namespace std;

int solution(vector<int> people, int limit) {
  sort(people.begin(), people.end());
  int count = 0;
  int i = 0;
  int j = people.size() - 1;

  while (i <= j) {
    if (people[j] + people[i] <= limit) {
      i++;
    }
    j--;
    count++;
  }

  return count;
}
```
