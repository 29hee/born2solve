**오늘의 메모(선택)**: 
파이썬은 split 딸깍인데...

Change는 출력하지는 않지만, 닉네임이 바뀌어야 한다.

**코드(선택)**:

```python
from typing import List, Dict

def solution(record: List[str]) -> List[str]:
    answer: List[str] = []
    user_nickname: Dict[str, str] = {}
    
    for line in record:
        cmd = line.split(" ")
        action = cmd[0]
        uid = cmd[1]
        
        if action in ("Enter", "Change"):
            nickname = cmd[2]
            user_nickname[uid] = nickname
            
    for line in record:
        cmd = line.split(" ")
        action = cmd[0]
        uid = cmd[1]
        
        if action == "Enter":
            answer.append(f"{user_nickname[uid]}님이 들어왔습니다.")
        elif cmd[0] == "Leave":
            answer.append(f"{user_nickname[uid]}님이 나갔습니다.")
    
    return answer
```
