**오늘의 메모(선택)**: 
1. 포켓몬 n 마리 중에 n/2 마리, 종류를 가장 다양하게 선택해야 한다.
2. 포켓몬 종류 번호의 개수를 return 하는 함수 구현
3. n/2가 종류보다 크면 종류의 수 선택
4. n/2가 종류보다 작다면 n/2를 선택

**코드(선택)**:

```python
def solution(nums):
    answer = 0
    set_pokemon = set(nums)
    if len(nums)/2 > len(set_pokemon):
        answer = len(set_pokemon)        
    else:
        answer = len(nums)//2
    return answer
```
