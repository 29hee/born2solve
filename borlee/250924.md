**코드(선택)**:

```python
import collections

class Solution:
    def numJewelsInStones(self, jewels: str, stones: str) -> int:
        stone_counts = collections.Counter(stones)
        jewel_count = 0
        for jewel in jewels:
            jewel_count += stone_counts[jewel]
        return jewel_count
```
