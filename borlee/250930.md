**오늘의 메모(선택)**: 
간간이 하는 코딩 기초 트레이닝 문제

**코드(선택)**:

```python
from typing import List

def solution(intStrs : List[str], k : int, s : int, l : int) -> List[str]:
    answer: List[str] = []
    for string in intStrs:
        part : str = string[s: s+l]
        num : int = int(part)
        if num > k:
            answer.append(num)
    return answer
```
