**오늘의 메모(선택)**: 
코딩기초문제
리스트 슬라이싱 사용[:]~ -> 정렬

**코드(선택)**

```python
from typing import List

def solution(my_string: str) -> List[str]:
    suffix_arr = [my_string[i:] for i in range(len(my_string))]
    return sorted(suffix_arr)
```
