**문제 링크**: https://school.programmers.co.kr/learn/courses/30/lessons/140107

**오늘의 메모(선택)**: 
                여러분들과 같이 푸는 바람에 머릿 속에서 각인이 되었어요. 

**코드(선택)**:

C++

```

#include <string>
#include <vector>
#include <cmath>

using namespace std;

long long solution(int k, int d) {
    
    long long answer = 0;
    long long bigD = d;
    for (long long i = 0; i <= d; i += k)
    {
        int y = sqrt((bigD * bigD) - (i * i));
        answer += y / k + 1;
    }
    return answer;
}

```
