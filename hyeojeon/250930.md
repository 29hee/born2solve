**문제 이름 또는 번호**: 2 x n 타일링

**문제 링크**: https://school.programmers.co.kr/learn/courses/30/lessons/12900

**오늘의 메모(선택)**: dp최고

**코드(선택)**:

```cpp

#include <string>
#include <vector>

using namespace std;

int solution(int n) {
    vector<int> dp(n+1);
    
    dp[0] = 1;
    dp[1] = 1;
    
    for (int i = 2; i <= n; i++)
        dp[i] = (dp[i-1] + dp[i-2]) % 1000000007;
    return dp[n];
}


```
