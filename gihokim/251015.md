**문제 링크**: https://leetcode.com/problems/length-of-last-word/description/

**오늘의 메모(선택)**: 하하...17 (물씬 풍기는 피신의 향기)

**코드(선택)**:

```c++
class Solution {
public:
    int lengthOfLastWord(string s) {
        int st_i;
        int end_i = s.length() - 1;

        while (s[end_i] == ' ')
            end_i--;

        st_i = end_i;
        
        while (st_i > 0 && s[st_i - 1] != ' ')
            st_i--;

        return (end_i - st_i + 1);
    }
};
```
