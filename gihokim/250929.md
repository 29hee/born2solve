**오늘의 메모(선택)**: 하하...13 (문제가 말만 번지르르 하지 하라는건 간단하다)

**코드(선택)**:

```c++
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        for (vector<int>::iterator it = nums.begin(); it != nums.end();)
            it = (*it == val) ? nums.erase(it) : it + 1;
        return (int)nums.size();
    }
};
```
