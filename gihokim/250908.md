**문제 이름 또는 번호**: 2884 / 알람 시계

**문제 링크**: https://www.acmicpc.net/problem/2884

**오늘의 메모(선택)**: 하하..

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

int main() {
	int hour, minute;
	cin >> hour >> minute;

	minute -= 45;

	if (minute < 0) {
		hour--;
		minute += 60;
	}
	
	if (hour < 0) {
		hour += 24;
	}

	cout << hour << ' ' << minute;
}
```
