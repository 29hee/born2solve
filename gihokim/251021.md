**문제 이름 또는 번호**: 69. Sqrt(x)

**문제 링크**: https://leetcode.com/problems/sqrtx/description/

**오늘의 메모(선택)**: 에라토스테네스의 체가 뭐였는지 기억이 안나서 그냥 반복문 돌림

**코드(선택)**:

```c++
class Solution {
public:
    int mySqrt(int x) {
        for (int n = 1; n <= 46341; n++) {
            if ((long)n * n == (long)x)
                return n;
            else if ((long)n * n > (long)x)
                return n - 1;
        }
        
        return 21e8; // never reach here
    }
};
```
