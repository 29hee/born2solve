**문제 이름 또는 번호**: 2675 / 문자열 반복

**문제 링크**: https://www.acmicpc.net/problem/2675

**오늘의 메모(선택)**: 하하...8

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

int T;

int main() {
	cin >> T;

	for (int tc = 1; tc <= T; tc++) {
		int n;
		string str;

		cin >> n;
		cin >> str;

		for (int i = 0; i < str.length(); i++) {
			for (int j = 0; j < n; j++) {
				cout << str[i];
			}
		}
		cout << '\n';
	}
}
```
