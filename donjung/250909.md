**오늘의 메모(선택)**: 
연속된 세 개의 자연수 중에는 반드시 3의 배수가 아닌 수가 존재.
이를 통해 숫자를 찾고, 그로부터의 상대적 위치를 이용한 다음 숫자를 추론하여 규칙을 적용한다.

**코드(선택)**:
``` c++
#include <iostream>
#include <string>
#include <vector>

int try_stoi(const std::string& s) {
	try {
		return std::stoi(s);
	} catch (...) {
		return -1;
	}
}

int main() {
	std::ios_base::sync_with_stdio(false);
	std::cin.tie(NULL);

	std::string s1, s2, s3;
	std::cin >> s1 >> s2 >> s3;

	int next_num = 0;
	int num_val = -1;

	// 어떤 입력이 숫자인지 확인
	num_val = try_stoi(s1);
	if (num_val != -1) {
		next_num = num_val + 3;
	} else {
		num_val = try_stoi(s2);
		if (num_val != -1) {
			next_num = num_val + 2;
		} else {
			num_val = try_stoi(s3);
			next_num = num_val + 1;
		}
	}

	if (next_num % 15 == 0) {
		std::cout << "FizzBuzz" << std::endl;
	} else if (next_num % 3 == 0) {
		std::cout << "Fizz" << std::endl;
	} else if (next_num % 5 == 0) {
		std::cout << "Buzz" << std::endl;
	} else {
		std::cout << next_num << std::endl;
	}

	return 0;
}

```
