**문제 플랫폼**: 백준

**문제 이름 또는 번호**: 한수

**문제 링크**: https://www.acmicpc.net/problem/1065

**오늘의 메모(선택)**:

**코드(선택)**:
```cpp
#include <iostream>
#include <vector>

int main()
{
	int N;
	std::cin >> N;

	if (N < 100)
	{
		std::cout << N;
		return 0;
	}

	int result = 99;
	for (int i = 100; i <= N; ++i)
	{
		
		std::vector<int> vec;
		std::string str = std::to_string(i);
		for (int j = 0; j < str.size(); ++j)
			vec.push_back(str[j] - '0');

		int gap = vec[1] - vec[0];
		bool isHansu = true;
		for (int j = 1; j < vec.size(); ++j)
		{
			if (gap != vec[j] - vec[j - 1])
			{
				isHansu = false;
				break ;
			}
		}
		if (isHansu == true)
			++result;
	}

	std::cout << result;

	return 0;
}
```
