**문제 이름 또는 번호**: 문자열 반복

**문제 링크**: https://www.acmicpc.net/problem/2675

**오늘의 메모(선택)**: ㅎㅎ..

**코드(선택)**:
```cpp

#include <iostream>

int main()
{
	int n;
	std::cin >> n;

	int R;
	std::string S;
	while (n--)
	{
		std::cin >> R >> S;

		for (int i = 0; i < S.size(); ++i)
		{
			for (int j = 0; j < R; ++j)
				std::cout << S[i];
		}
		std::cout << std::endl;
	}

	return 0;
}

```
